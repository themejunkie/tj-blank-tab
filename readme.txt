==Description==

This plugin will add target="_blank" functionality in comment and post content, which open the link with a new window(Tab).

==Installation==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `TJ Blank Tab` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3.You have done.
